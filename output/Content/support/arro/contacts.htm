Contacts

The Importance of Relationships with Contacts
Income may be received through your relationships. In order to maximise the engagement and contributions from your contacts, you need to understand and profile the different types of relationships you have with them. You will have contacts to whom you are accountable. In other words you will be expected to contribute towards these bodies’ aims. You will have other contacts with whom you conduct transactions i.e. you will provide goods or services and in return they will give you cash, equipment, resources, facilities or materials. Finally you will have contacts who are looking for a deeper level of engagement and participation in your school than simply giving you money. These are known as developmental contacts who will want to be involved in the planning and delivery of your projects and activities.

 

The Contacts module is where you manage all your contact details.

Create a contact

You can add contacts manually here in the Contacts module. If you have a lot of contacts you want to add as a batch, you can import these from a csv file, provided it is set up correctly. For more information, see Import Data.

Note: If your school SIMS is set up to integrate with Arro, then your parent contact details will be automatically pulled into Arro and shown in the list of contacts.

  1. Open Supporters > Contacts.
  2. Click + Add Contact > Create manually.
  3. Select the Contact Type from the drop-down list.

    4. This helps groups contacts in Arro making it easier for you to send campaigns to different groups.

  4. Enter the contact name, postal and email details.
    • If the contact has a Twitter account, add this in the Twitter box.

      • Tip: You can add the Twitter account with or without the @ preceding the Twitter handleThe Twitter user name identifying the Twitter user on Twitter..

  5. Click Create.

Amend a contact

You can amend any contact details required.

  1. Open Supporters > Contacts.
  2. Select the contact name to amend.

    3. Tip: Filter the list of names displayed by entering the name or characters from the name into the Search box.

  3. Click the Name to display the contact details.
  4. Make any changes you require.
  5. Click Save.

Marking contacts as in or out of use

By default when you create a contact, the contact is in use. You can mark a contact as out of use or back in use. This means that you can create and hold a full list of contacts even if people are occasionally not available to help you.

Mark contacts as out of use

  1. Open Supporters > Contacts.

  2. On the Active tab, select the contact name to mark out of use.

    3. Tip: Filter the list of names displayed by entering the name or characters from the name into the Search box.

    Note: You can select multiple names.

  3. Select Mark Inactive. The contact names move from the Active to the Inactive tab.

Mark contacts as active

  1. Open Supporters > Contacts.
  2. Select the Inactive tab.
  3. Select the names of the contacts you want to mark as active.
  4. Select Mark active.

Import Data

If you have a lot of contacts to set up you can do this in one go by importing the information from a CSV (comma separated values) file provided you set it up correctly.

You can create a Microsoft Excel spreadsheet with the extension .csv to hold the information you want to import. The first line in the file must be the header which specifies the fields you intend to import. Then each contact’s information must be entered on a new line in the csv file.

If you want to create a file of your own, you can do this by downloading the template from Contacts > + Add contact > Import > Download template.

Note: Records are assigned unique identifier numbers (IDs) in the import file. All records with unique IDs are appended to the existing list of contacts within Contacts. If the ID in the import file is not unique and matches one already in Contacts, then the existing record is overwritten. You can see the unique ID of existing records by exporting the stakeholder list.

Creating your own import file

  1. OpenSupporters > Contacts> + Add contact > Import.
  2. Click Download Template

    3. A contact template .xslx is downloaded to your PC.

  3. Open this template .xslx and save it to your PC with a name of your choosing and the Save as type CSV (comma delimited).
  4. Enter the details of the contacts you want to import.

    5. Remember the information is entered for each name on a separate line.

    e.g.

    Header

    Contact 1 information

    ….

    Contact N information

Importing data

  1. Open Supporters > Contacts > + Add contact > Import.
  2. If you have pending imports - imports that you have uploaded already but not imported yet - click View Pending and continue from step 6.
  3. Otherwise, click Browse to choose a file … and browse to select the .csv file you want to import.
  4. Select the file and click Open.
  5. Click Upload.

    6. You will see a message saying the file has uploaded a number of records successfully.

  6. A list of contacts to be imported is then displayed across two tabs: New Contacts and Updated Contacts.
    1. Select the tab.
    2. To remove records:
      • To remove all records from the import, click Remove All.
      • To remove selected records from the import, select the records you want to remove and click Remove Selected.
    3. To import records:
      • To import all records, click Import All.
      • To import selected records, select the records you ant to import and click Import Selected.

    Note: Records remain pending until they have all been imported or removed from both New Contacts and Updated Contacts.

Export Data

If you have your contacts in another system and you want to export the data from Arro to include in another system, you can export the data from Arrointo a spreadsheet which you can save as a csv file which you can then import into another system.

  • Open Supporters > Contacts > Export.

    • The contacts information is downloaded to your PC as a .xls file.
