Projects Pages

Projects Index Page

When you click on Projects at the top of the Joinos Community Site Home Page or View our projects mid-way down the Home Page, the Projects Index Page is displayed. This is where your community can see how many projects you have in progress and decide which they would like to help with.

Within the Projects module, you choose a featured project. This has the highest visibility on the Projects Index Page and its details are shown across the page header with the total raised for the featured project being displayed at the top right-hand side of the page.

Under the section, Our school projects, you can read all about the other projects.

How do you choose a featured project?

  1. Select the project you want to feature highest on your Joinos Community Site Project Index page.
  2. Click Choose Featured Project.

    3. The details for this project are shown in the header of the Projects Index page when you click Projects from the Joinos Community Site Home page.

Project Joinos Page

You can view a project’s Joinos page by:

  • Clicking Find out more alongside a project on the Project Index Page
  • Clicking View Joinos page from the Project Details page in Projects.

From the project’s Joinos page, users can easily:

  • Like the project.
  • Find out more about the project.
  • Get involved with the project.
  • See the donations that have been made to the project.
  • Spread the word about the project.

How do you create Joinos pages for your projects?

Each project’s Joinos page is created by publishing the Joinos page from the Project Details page in Projects.

To create a project Joinos page

  1. Open Projects and select the project name from the Active tab.

    2. A project Joinos page is created on the Joinos community site.

    Note: If this is the first project page you have created, the Joinos Community site is created for you automatically with details about the school and a number of pre-set News & Media items.

  2. To preview or view the project’s Joinos page:
    • If you have not yet published the project’s Joinos page, click Preview Joinos page.
    • If you have published the page previously, click View Joinos page.
  3. To publish the project’s Joinos page:
    • If you have not yet published the project’s Joinos page, click Publish.
    • If you have published the page previously, click Publish changes.

To edit project details on the project Joinos page

  1. Open Projects and select the project name from the Active tab.
  2. In the top section, click Edit.

    3. Note: The date the narrative was last edited is displayed below the Edit button.

  3. Amend the Project Name if required. This is the title of the project Joinos page.
  4. Amend the Project Description if required. This information is relayed as Our project story on the project Joinos page.

    5. You can add a link from the Project Description to other related web pages that relate to the project by wrapping your link in square brackets and inserting the link in normal brackets immediately afterwards. For example, [Click this link](www.example.com). This will display the text Click this link as a link in the published page and when you click on it, the web page www.example.com will open. You can open this in a new window or tab if you right-click on the link and select Open in a new window or Open in a new tab.

    Note: www.example.com is a fictitious URLAn address for an item on the Internet., used for example purposes only.

  5. Choose whether to save the changes you have made or undo them:
    • If you are happy with the changes you have made, click Done.
    • If you changed your mind about what you have written on this occasion and want to revert to the last saved version of the text, click Undo Changes.
  6. To change the image displayed on the project’s Joinos page, click Select image and click on the image to use.

    7. Tip: If you want to use your own image, remember to upload the image file to the project first, using the Files tab in Project Details in Projects to upload your own image to use on the project’s Joinos page.

  7. Click Publish changes.

Promote and share your project’s Joinos page

You need to promote and share your project’s Joinos page in order to make it work for your project.

You can:

  • Create a campaign about the project, using email or social media. For more information on creating and managing email campaigns, see Email Campaigns. For information on creating and managing social media campaigns, see Social Media Campaigns.
  • Share one-off social media messages about the project, using the Spread the word icons on the project’s Joinos page.

    • Note: You will need to log on and authorise the relevant app to share on your behalf.

  • Copy the Joinos page URLAn address for an item on the Internet. to include in an email or on your school website.
    • Highlight the URL.
    • Right-click and select Copy from the drop-down menu that appears.
